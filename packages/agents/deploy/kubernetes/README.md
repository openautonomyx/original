# k3s Deployment

Default deployment target: k3s.

## Assets

- `deployment.yaml`

## Use

- Deployment
- Service
- Ingress
- ConfigMap
- Secret

## Avoid For Now

- Helm
- service mesh
- complex operators
- multi-cluster setup
- heavy observability stack
- GitOps requirements

## Principle

```text
Default = k3s
Compatible = standard Kubernetes
Keep it lightweight
```
