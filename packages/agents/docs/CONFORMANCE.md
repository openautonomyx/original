# Publishing Platform Conformance

Required for publication:
- Valid Schema.org JSON-LD
- Author
- Publisher
- License
- Canonical URL
- Language
- Tenant ownership
- Revision history
- Provenance metadata
- Editorial workflow approval
- Publication timestamp

Workflow:
Draft -> Review -> Approved -> Published -> Archived

No content may be published without approval state.

Tenant isolation, auditability, traceability and revision history are mandatory.
