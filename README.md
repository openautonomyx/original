# OpenAutonomyx Monorepo

Unified repository containing all code, configuration, and documentation.

## 📦 Packages

- **platform/** - Main application
- **agents/** - Agent registry and core
- **publications/** - Content and publications
- **console/** - Deployment console and scripts

## 🚀 Quick Start

```bash
yarn install
yarn dev
```
